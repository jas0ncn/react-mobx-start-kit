# React Mobx Start Kit

Quick start with React, Mobx and TypeScript.

## Packages Including

 - react
 - react-router4
 - mobx
 - react-css-module
 - less-complier

## Commands

Install dependencies

```bash
npm i
```

Run as development environment

```bash
npm run dev
```

Build files for production

```bash
npm run build
```
